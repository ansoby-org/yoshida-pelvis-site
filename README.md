# 吉田式 骨盤体操教室 サイト

## ローカルでの開発・確認方法

ローカル環境でサイトを確認・変更するには、**Tailwind CSS の監視**と**Hugo サーバーの起動**の2つを同時に行う必要があります。

### 1. 初回セットアップ

依存パッケージ（Tailwind CSS等）をインストールします。

```bash
npm install
```

### 2. サーバーの起動

ターミナルを2つ開き、それぞれで以下のコマンドを実行してください。

**ターミナル1: CSSの監視ビルド**
Tailwind CSS の変更を監視し、リアルタイムで `main.css` を生成します。

```bash
npm run tw:dev
```

**ターミナル2: Hugo サーバーの起動**
Webサーバーを立ち上げます。
`-b /` オプションは、ローカル環境でルートパス (`http://localhost:1313/`) で正しく表示させるために必要です。

```bash
hugo server -b /
```

### 3. ブラウザで確認

以下のURLにアクセスしてください。

http://localhost:1313/
